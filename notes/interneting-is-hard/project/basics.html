Headings

Headings define the outline of your site. There are six levels of headings.

Lists

Unordered list grouped items without a specific order:

  • Add a "ul" element (it stands for unordered list)
  • Add each item in its own "li" element
  • They don't need to be in any particular order

Ordered list displays items in a specific sequence like recipe steps:

  1. Notice the new "ol" element wrapping everything
  2. Also note how the numbers increment on their own
  3. But, the list item elements are the same
  4. You should be noticing things is this precise order, because this an ordered list
